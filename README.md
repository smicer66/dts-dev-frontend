
# HMCTS

The HMC Frontend provides a web interface for caseworkers to manage their tasks efficiently. It connects to the backend API to display, create, and update tasks, enabling a seamless user experience.
This project is built using Node.js, Express, Handlebars, and JavaScript and is the front component of the application.
The backend of the application can be found at https://github.com/smicer66/dts-dev.git
## Features

[Features]
List all tasks assigned to caseworkers
Add new tasks through web forms
Update existing tasks
Dynamic views with Handlebars templates
Connects to the backend API for data persistence


## Tech Stack
Backend Framework: Node.js + Express
Templating: Handlebars
Styling: CSS / SCSS / Less
HTTP Requests: Request
## Setup & Installation
Prerequisites
Node.js v18 or above
npm
Steps
Clone the repository
git clone https://github.com/smicer66/dts-dev-frontend.git
cd dts-dev-frontend
Install dependencies
npm install
Run the application
npm start
Access the frontend

The application runs by default on:

http://localhost:3000


## Usage
Navigate to http://localhost:3000
Interact with tasks: list, create, update
Ensure the backend API is running (https://github.com/smicer66/dts-dev
)
